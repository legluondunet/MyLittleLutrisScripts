# Bad Mojo - CD
### Description:
Play "Bad Mojo" CD edition on Linux!
### Technical notes:
- You will need install CD only one time during install
- This game will launch in a window 640x480. During my tests, the only solution to play this game in fullscreen was to use gamescope
### Credits:
- Lutris script maintained by legluondunet: https://github.com/legluondunet/MyLittleLutrisScripts


# Bad Mojo - CD + Windows 3.1
### Description:
Play "Bad Mojo" CD edition on Linux!
### Technical notes:
- Using Dosbox-Staging with a ready to use Windows 3.1 install
- Once Windows 3.1 started, in "Main" group click on "Filemanager", click on "D" drive and install the game
### Credits:
- Dosbox-Staging: https://www.dosbox-staging.org
- Lutris script maintained by legluondunet: https://github.com/legluondunet/MyLittleLutrisScripts
