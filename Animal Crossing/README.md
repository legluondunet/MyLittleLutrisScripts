## Animal Crossing - ACGC-PC-Port
### Description:
Play "Animal Crossing" on Linux
### Technical notes:
- Using ACGC-PC-Port engine
- During install, you will need  to provide Animal Crossing Gamecube rom file (USA - SHA1 = 2d2b1fa3883f49af779ce9ca133db3be17be8f32)
