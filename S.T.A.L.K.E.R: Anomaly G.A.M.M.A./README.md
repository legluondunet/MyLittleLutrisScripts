# S.T.A.L.K.E.R: Anomaly G.A.M.M.A. - Launcher Clone
### Description:
Play "S.T.A.L.K.E.R: Anomaly G.A.M.M.A." on Linux
### Technical notes:
- Using Stalker GAMMA Launcher Clone, a clone of Grokitach's Stalker GAMMA Launcher with WINE compatibility and extra features
- This script installer follow FaithBeam guide steps available on https://github.com/FaithBeam/stalker-gamma-launcher-clone/wiki/Linux-Install
- You will need 150 Go hard disk space
- When you will launch "Stalker GAMMA Launcher Clone" for the first time, on the left panel, when highlighted, click on: 
1) Downgrade ModOrganizer
2) First install initialization
3) Full Install
4) Play
