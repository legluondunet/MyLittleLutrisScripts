# MDK - CD + dgVoodoo2
### Description:
Play "MDK" CD edition on Linux!
### Technical notes:
- During install let all options to default
- Available languages: English, French, Spanish, Polish, Italian
- This game does not support ALT-TAB. Workaround: use Gamescope
