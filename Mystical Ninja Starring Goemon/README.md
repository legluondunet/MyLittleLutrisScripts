## Mystical Ninja Starring Goemon - Goemon64Recomp AppImage
### Description:
Play "Mystical Ninja Starring Goemon" on Linux!
### Technical notes:
- Using Goemon64Recomp engine
- You will need to provide "Mystical Ninja Starring Goemon" USA rom file in .z64 format
