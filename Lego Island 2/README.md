# Lego Island 2 - CD + dgVoodoo 2
### Description:
Play "Lego Island 2" CD edition on Linux!
### Technical notes:
- Game executable already patched with "Load screen fix" and "Skip cinematics"
- Use xdotool to work around a problem with Wine where a black window appears in front of the game window.
