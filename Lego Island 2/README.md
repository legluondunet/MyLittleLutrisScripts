# Lego Island 2 - CD + dgVoodoo 2
### Description:
Play "Lego Island 2" CD edition on Linux!
### Technical notes:
- Game executable already patched with "Load screen fix" and "Skip cinematics"
- Use xdotool to work around a Wine issue where a black window appears in front of the game window. If this happens, simply press ALT+TAB to select the game window

- dgVoodoo2: https://github.com/dege-diosg/dgVoodoo2
- Lutris script maintained by legluondunet: https://github.com/legluondunet/MyLittleLutrisScripts
