# Scorched Planet - CD Windows
### Description:
Play "Scorched Planet" CD Windows edition on Linux!
### Technical notes:
- Using DxWnd and dgVoodoo2 for better graphic stability
- Known issue: regularly crashing when launching a new game or load a save
