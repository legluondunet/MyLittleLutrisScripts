# Warlords Battlecry 3 - GOG + cnc-draw
### Description:
Play "Warlords Battlecry 3" GOG edition on Linux!
### Technical notes:
- Using cnc-draw wrapper for better graphic rendering and stability
- Known issue: during my tests, this game had graphics issues with Proton Wine version, units was invisible. Workaround: use a recent Wine-Staging version
