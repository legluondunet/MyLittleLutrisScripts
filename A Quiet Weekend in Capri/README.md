# A Quiet Weekend in Capri - CD + Windows 98
### Description:
Play "A Quiet Weekend in Capri" CD edition on Linux!
### Technical notes:
- Using Dosbox Pure Unleashed with a ready to use Windows 98 SE install
- How to install and play, follow this steps:
1) Once Windows 98 started, go to "My Computer", CD Drive, install the game
2) Once game installed, change resolution to 1024x768 using monitor icon located on the down right
3) Change CD, press "CTRL+F12" keys and choose dev-capb.cue
4) Click on Windows start menu, programs, Oxiana, Capri, launch the game
