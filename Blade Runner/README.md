# Blade Runner - CD - ScummVM
### Description:
Play "Blade Runner" CD edition on Linux!
### Technical notes:
- You will need CDs only one time during install
- English, French, German, Italian and Spanish subtitles included
### Credits:
- ScummVM subtitles for Blade Runner: https://www.scummvm.org/games/#addons-bladerunner:bladerunner
- Lutris script maintained by legluondunet: https://github.com/legluondunet/MyLittleLutrisScripts
