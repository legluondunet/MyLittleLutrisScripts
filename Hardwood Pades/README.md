# Hardwood Pades - 2.1 Build 61 + Windows 95
### Description:
Play "Hardwood Spades" on Linux!
### Technical notes:
- Using Dosbox Pure Unleashed with a ready to use Windows 95 OSR 2.5 install
- Once Windows 95 started, go to "My Computer", click on "D" Drive and install the game
- Press "CTRL+F12" to show Dosbox Pure Unleashed menu
