# Harry Potter and the Goblet of Fire - DVD + Chip-Biscuit wrapper
### Description:
Play "Harry Potter and the Goblet of Fire" DVD edition on Linux!
### Technical notes:
- You will need to provide game DVD image in ISO format
- Using Chip-Biscuit wrapper for better graphic rendering, stability
- Known issue: cinematics are distorted if you use an aspect ratio other than 4:3
