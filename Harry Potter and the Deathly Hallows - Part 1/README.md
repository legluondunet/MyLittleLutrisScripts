# Harry Potter and the Deathly Hallows - Part 1 - DVD + Chip-Biscuit wrapper
### Description:
Play "Harry Potter and the Deathly Hallows - Part 1" DVD edition on Linux!
### Technical notes:
- Using Chip-Biscuit wrapper for better graphic rendering and stability
- Known issue:
* Using ALT+TAB during cutscenes can cause significant game instability
