# Tex Murphy - Under a Killing Moon - GOG
### Description:
Play "Tex Murphy - Under a Killing Moon" provided by GOG on Linux!
### Technical notes:
- Using tex3 mouselook mod:
    - Mouse - Look around
    - [W] - forwards
    - [A] - strafe left
    - [S] - backwards
    - [D] - strafe right
    - [C] - crouch (while held)
    - [R] - reach up on tippytoes (while held)
    - [L-Shift] - run (while held)
