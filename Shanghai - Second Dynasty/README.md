## Shanghai - Second Dynasty - CD + Windows 98
### Description:
Play "Shanghai - Second Dynasty" on Linux!
### Technical notes:
- Using Dosbox Pure Unleashed with a ready to use Windows 98 SE install
- Once Windows 98 started, go to "My Computer", CD Drive and install the game
- Set Windows 98 video resolution to 1024x768 to be able to install this game in "High Resolution"
- Press "CTRL+F12" to show Dosbox Pure Unleashed menu
