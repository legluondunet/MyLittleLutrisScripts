# Unreal Gold - CD + OldUnreal patch
### Description:
Play "Unreal Gold" CD edition on Linux!
### Technical notes:
- Using CD from Internet Archive with OldUnreal patch 227k_13
- You can choose to play the Windows (x64) or Linux (x64) versions
- This script installer will automagically set the game to your desktop resolution
- Press F1 key to show console
- Multilingual: you will be able to choose your preferred language for menus and subtitles during installation between English (default), French, German, Italian, Spanish, Catalan, Dutch, Polish, Portugues and Russian
- This game does not natively support controller, you can download a AntiMicroX profile here: 
https://github.com/legluondunet/MyLittleLutrisScripts/tree/master/Unreal/gamepad%20profiles
