HERETIC II v1.07 FAQ/Known issues



  • Heretic II crashes when I begin a new game (easy, medium, hard)


Heretic II will crash if you launch directly a new game, it’s a know issue for this game in Windows.

That crash at the beginning of the game could be caused by new CPU that load too quickly the game data.

Gho, DxWnd dev said  about this crash:

« the game tries to run a procedure at address 0, as if it was using a function pointer that it didn't have time to load yet. »

Workaround: Launch the tutorial level first, escape to the main menu, start a new game.

This bug only occurs if you launch a new game directly, not when you load or save a game.


  • No soundtrack ingame?


Normally the Lutris script would have to rip the game soundtrack during the installation.

If this is not the case, if you do not hear the music during the game, rip cd audio tracks from your Heretic II CD to ogg or flac format in separate files.

Call each track "track01.ogg", "track02.ogg" or "track01.flac", "track02.flac"etc... and put them in the main Heretic II game folder:

c:\Program Files (x86)\Heretic II\music

Launch the game with Lutris you can now hear Heretic II soundtrack during gaming.


  • The game does not launch anymore


You tried to change some video options and the game does not launch anymore?

Delete the config.cfg file located in this folder:

c:\Program Files (x86)\Heretic II\user

Now the game should launch normally again.


  • How to play multiplayer game

Open Heretic II console (press ~ key) and type :

connect IP_server

You will find IP servers on this site :

https://hexenworld.org/qstat/

and you can ask for support, multiplayer games and more on Heretic II Community Hub Facebook group :
