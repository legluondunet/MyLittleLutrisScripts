Gamepad profiles for Enclave

Enclave have very weak support for gamepad on modern computer, you have to install a third party software.

This software will attribute keyboard keys to your gamepad.

Depend your gamepad, choose appropriate app below, install it on your Linux distribution, import profile and start game, you can now play with your gamepad.

x360 controller profile to use with antimicro --> enclave_x360.gamecontroller.amgp

You can download Antimicro here: https://github.com/AntiMicro/antimicro

Steam controller profile to use with sc-controller --> enclave_steam.sccprofile

You can download sc-controller here --> https://github.com/kozec/sc-controller

You will find gamepad profiles in your Lutris Enclave wineprefix: c:\GOG Games\Enclave\gamepad profiles\
