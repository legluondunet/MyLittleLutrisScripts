# The Multimedia Bug Book - CD - Windows 95 OSR 2.5
### Description:
Play "The Multimedia Bug Book" on Linux!
### Technical notes:
- Using Dosbox Pure Unleashed with a ready to use Windows 95 OSR 2.5 install
- Once Windows 95 started, go to "My Computer", CD Drive and install the game using setup.exe file. 
If Quicktime does not work, reinstall it with installer available on desktop
- Press "CTRL+F12" to show Dosbox Pure Unleashed menu
