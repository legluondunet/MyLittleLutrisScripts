# Need for Speed: Most Wanted - Black Edition - DVD
### Description:
Play "Need for Speed - Most Wanted - Black Edition" DVD edition on Linux!
### Technical notes:
- Included update patch 1.3
- Using DxWrapper for better graphic stability and ThirteenAG widescreen fix
- Per default game  resolution set to 1280x720. You can change game resolution in "Video options"
- Available languages for this game: Danish, German, English, Spanish, French, Finnish, Italian, Korean, Dutch, Polish, Swedish and Chinese (Simplified and Traditional)
